## 1. 사전준비(uv package 사용)
- Python3.11.9
- uv
- git
- gstreamer
### Step1 : install uv
In your terminal, run:
```shell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```
Verify installation:
```shell
uv --version
```

### Step2 : install Python
In your terminal, run:
```shell
uv python install 3.12 --default
```

### Step3 : Install Git
Download and install Git for Windows
	https://git-scm.com/install/windows

Finally, Git LFS then needs to be initialized with the command :
``` shell
git lfs install
```

Install GStreamer using the official installer
	https://gstreamer.freedesktop.org/download/
	
### Step4 : make project directory
In your terminal, run:
```shell
mkdir reachy_mini_project
cd reachy_mini_project
```

### Step5 : git clone 
```shell
git clone https://github.com/orocapangyo/reachy_mini.git
```

### Step6 : Create the environment (using uv)
```shell
uv venv reachy_mini_env --python 3.12
```

### Step7 : Install Reachy Mini & MuJoCo
```shell
uv pip install "reachy-mini[mujoco]"
```

### Step8 : Run MuJoCo Simulation
```shell
reachy-mini-daemon --sim
```
