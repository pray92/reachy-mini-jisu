# 동영상 녹화

### 1. 녹화 코드: "로봇의 순간순간을  저장"

로봇이 움직이는 매 순간을 관찰해서 저장.

- **관찰 (Data Capture):** `mini.get_current_joint_positions()`를 호출하면 로봇이 "현상태 각도는 , 안테나는 상태.
- **정리 (Structuring):**  **`head` (머리 7개)**와 **`antennas` (안테나 2개)*
- **저장 (Saving):** 5초 동안 찍은 수많은 사진(데이터)을 `clean_motion.json`이라는 하나의 파일로 묶어서 저장

import time
import json
import random
from reachy_mini import ReachyMini

with ReachyMini() as mini:
duration = 5  # 녹화 시간(초)
hz = 30       # 초당 프레임 수
trajectory = []

```
print("수정된 구조로 녹화 시작 (로봇을 움직여주세요)...")
start_time = time.time()

while time.time() - start_time < duration:
    # --- [옵션] 로봇에게 테스트 동작 부여 (마우스로 움직일 경우 생략 가능) ---
    if len(trajectory) % 15 == 0:
        mini.set_target_antenna_joint_positions([random.uniform(-0.3, 0.3), random.uniform(-0.3, 0.3)])
    # -------------------------------------------------------------------

    # 현재 상태 읽기
    # get_current_joint_positions()가 반환하는 [[7개], [2개]] 구조에서 알맹이만 추출
    current_raw_joints = mini.get_current_joint_positions()

    # 데이터 구조화: 나중에 꺼내기 쉽게 이름을 붙여 저장합니다.
    frame_data = {
        "timestamp": time.time() - start_time,
        "head": current_raw_joints[0],      # 머리 관절 7개 (리스트)
        "antennas": current_raw_joints[1]   # 안테나 관절 2개 (리스트)
    }

    trajectory.append(frame_data)
    time.sleep(1/hz)

# JSON 저장
with open("clean_motion.json", "w") as f:
    json.dump(trajectory, f, indent=4)

print(f"녹화 완료! 총 {len(trajectory)} 프레임이 'clean_motion.json'에 저장되었습니다.")

```

### 2. 재생 코드:

저장된 파일을 다시 읽어서 로봇에게 명령.

- **읽기 (Loading):** 위에서 저장한`clean_motion.json` 파일을 열어 첫 번째 줄부터 읽기 시작.
- **분리 전달 (Execution):** * `head` 데이터를 꺼내 머리 모터 제어.
    - 동시에 `antennas` 데이터를 꺼내 안테나 모터들에게 명령을 보냅니다.제어
- **박자 맞추기 (Sync):** `time.sleep(1/30)`을 통해 녹화할 때와 똑같은 속도(1초에 30번)로 동작이

def play_clean_motion(file_path):
with open(file_path, 'r') as f:
trajectory = json.load(f)with ReachyMini() as mini:
    print("깔끔한 구조로 재생 시작...")
    for frame in trajectory:
        # 저장할 때 사용한 Key 이름 그대로 접근
        mini._set_joint_positions(frame['head'])
        mini.set_target_antenna_joint_positions(frame['antennas'])
        time.sleep(1/30)
    print("재생 완료.")

play_clean_motion("clean_motion.json")
